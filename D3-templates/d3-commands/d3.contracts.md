# /d3.contracts - Generate API Contracts

## Execute the Script

**On Linux/Mac:**
```bash
bash scripts/bash/d3-contracts.sh "d3-features/001-feature-name"
```

**On Windows:**
```powershell
. scripts/powershell/d3-contracts.ps1 "d3-features/001-feature-name"
```

## What This Does

1. Validates that plan.md exists
2. Creates `contracts/api.md` file
3. Provides structure for:
   - API endpoint definitions
   - Request/response specifications
   - Error handling contracts
   - Event definitions
   - Rate limiting rules
   - Authentication requirements

## Output

Creates file: `d3-features/001-feature-name/contracts/api.md`

## How to Use

1. Run `/d3.contracts` to generate the template
2. Define all API endpoints (GET, POST, PUT, DELETE)
3. Specify request parameters and response formats
4. Document error cases and responses
5. Define any events the system publishes
6. Specify authentication and rate limiting

## When to Run

- After the implementation plan is created
- Before starting backend implementation
- When designing the system's external interfaces
