# /d3.constitution - Project Principles & Governance

## Execute the Script

**On Linux/Mac:**
```bash
bash scripts/bash/d3-constitution.sh "project principles description"
```

**On Windows:**
```powershell
. scripts/powershell/d3-constitution.ps1 "project principles description"
```

## What This Does

1. Creates or updates `memory/d3-constitution.md`
2. Establishes project governing principles
3. Defines development guidelines
4. Sets governance rules for architectural decisions

## Output

Creates file: `memory/d3-constitution.md`

## Use Cases

- **New project**: Define the project's core principles
- **Existing project**: Update principles as team evolves
- **Architecture decisions**: Reference constitution when making design choices

## When to Run

- At the start of a new project
- When establishing team standards
- Whenever project principles change
